Project Overview

This interactive engineering portal tracks our technical design and physical creation of the Jiugang Bridge (舊港大橋) scale model in Hsinchu. Facing complex structural challenges, our team focused on wind resistance, weight distribution, and high-fidelity aesthetic modeling.

Engineering Lifecycle Timeline

Phase 01

Discovery & Initial Selection

Surveyed Old Port Island topography. Found two bridge channels and started analyzing the architectural layout of Baidi Bridge.

Establishing a concrete project focus. Collaboration was critical to map objectives before engineering.
Phase 02

Geometric Pivot Matrix

Measured Baidi Bridge scale properties. Found length-to-width ratios unsuitable for model building, leading to a pivot toward Jiugang Bridge.

Real-world metrics determined the original bridge choice was structurally impractical for a table prototype.
Phase 03

Foundational Towers & Core Anchors

Built the fundamental bridge platform, erecting structural support columns and main cable pylons.

Provides the load-bearing foundation needed to withstand weight from secondary bridge deck elements.
Phase 04

Color Separation Engineering

Transferred framework to work lab for surface finishing. Applied targeted paints to visually separate architectural components.

Improves visual clarity during data reviews while providing a clean, professional finish.
Phase 05

Main Runway Integration

Balanced and permanently bonded the main vehicle deck across the load pillars.

The deck is the primary platform; secure alignment ensures correct load paths and path structure.
Phase 06

Cabling & Environmental Rendering

Configured structural high-tension wires, safety guards, and shore topography using green sponges.

Increases physical model accuracy compared to the real Hsinchu site for better presentation engagement.
Phase 07

Empirical Loading Tests

Loaded micro-weights across different segments of the bridge structure to check for deflection or faults.

Isolates structural stress weaknesses, allowing us to implement reinforcement before final exhibition.
Phase 08

Exhibition Deployment

Compiled technical field data into public display layouts and practiced presentation delivery.

Translates raw structural engineering calculations into clear concepts for evaluators.
